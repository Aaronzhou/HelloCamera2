# HelloCamera2
Android 自定义相机 ： 可定制的UI, 裁剪指定区域, 大图预览, Camera / Camera2 API 的使用. 

在 Android Lollipop 及以后版本的 Android 中, 使用 Camera2 API. 在 Android Kitkat 及以前版本的 Android 中, 使用 Camera API.

详见 Sample App 的代码及注释.
